Mapping Source

This structure represents the following mapping definition:

obf.Substance maps to Substance:
  Language maps to language
  ImplicitRules maps to implicitRules
  Metadata.VersionId maps to meta.versionId
  Metadata.SourceSystem maps to meta.source
  Metadata.Profile maps to meta.profile
  Metadata.LastUpdated maps to meta.lastUpdated
  Metadata.SecurityLabel maps to meta.security
  Metadata.Tag maps to meta.tag
  Narrative maps to text
  Identifier maps to identifier
  Code maps to code
  Status maps to status
  Category maps to category // Change code system for extensibly bound codes from "http://hl7.org/fhir/substance-category" to "http://terminology.hl7.org/CodeSystem/substance-category"
  CommentOrDescription maps to description
  Ingredient maps to ingredient
  Ingredient.IngredientStrength maps to ingredient.quantity
  Ingredient.SubstanceOrCode maps to ingredient.substance[x]

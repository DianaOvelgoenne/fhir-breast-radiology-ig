Mapping Source

This structure represents the following mapping definition:

shr.core.RelatedArtifact maps to RelatedArtifact:
  Type maps to type
  LabelText maps to extension
  DisplayText maps to display
  Citation maps to citation  // string in R3, markdown in R4
  Url maps to url
  Attachment maps to document
  CanonicalUrl maps to extension  // resource is Ref(Any) in R3, but CanonicalUrl in R4
